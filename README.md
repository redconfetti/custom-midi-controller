# Custom MIDI Controller

![MIDI Controller Mockup]

[MIDI Controller Mockup]: /assets/midi-controller.png "MIDI Controller Mockup"

* Controls
  * 24 potentiometers
  * 24 momentary push buttons
  * Total: 48 inputs
* MIDI CC#
  * Knobs: 0 - 23
  * Buttons: 24 - 47
